🛡️ Level Up Your Linux Skills

You’ve mastered the basics—now it's time to apply your skills in real-world scenarios. These medium-level challenges are designed to simulate system administration, threat hunting, and data forensics using the Linux terminal.

Prepare to dig deeper into file permissions, log analysis, user management, and system monitoring. Good luck, cyber adventurer!

🕵️ Challenge 1: Log Hunt

A breach attempt has been logged in the system. You must find the log entry that matches the attack signature.

Use: grep, less, cat

👥 Challenge 2: Rogue User Cleanup

A rogue user account was added without authorization. Find and remove the account before it escalates privileges.

Use: cat /etc/passwd, sudo userdel

🔐 Challenge 3: Permission Panic

Sensitive files are exposed due to misconfigured permissions. Fix the access levels before data is leaked.

Use: chmod, ls -l

📊 Challenge 4: Disk Space Dilemma

The system is running low on space. Identify the largest directories and take corrective action.

Use: du -sh *, df -h, rm

🧬 Challenge 5: Scripted Escape

An automated script went rogue. Locate and analyze the script to neutralize the threat.

Use: find, cat, bash
