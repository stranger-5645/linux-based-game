üìò What is Linux?

Linux is an open-source operating system based on Unix. It is widely used in servers, development environments, cybersecurity, and even your Android phones. This platform will teach you to interact with Linux using a terminal.

The terminal (or shell) allows users to perform tasks using commands instead of clicking around with a mouse. These challenges will guide you step-by-step through core Linux skills. No experience needed!

üß† Command Knowledge

  • ls: List files and directories
  • cd: Change the current directory
  • pwd: Show current directory path
  • mkdir: Create a new directory
  • touch: Create a new file
  • cat: View file content
  • rm: Remove files or directories
  • cp: Copy files and directories
  • tree: Display directory structure
  • sudo: Run a command with elevated permissions
